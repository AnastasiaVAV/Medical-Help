Medical Help – медицинское приложение, состоящее из трех страниц:
– главная
- личный кабинет (если пользователь авторизован)
- контакты

## Установка и запуск
Все команды выполняются в корне проекта.

1. Установка зависимостей:
    ```bash
    make install
    ```
2. Запуск:
    ```bash
    make run
    ```
